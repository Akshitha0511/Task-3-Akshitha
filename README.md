The ATM Interface project is a Java console application that simulates the basic operations of an Automated Teller Machine. It allows users to create accounts, log in securely using a PIN, deposit and withdraw money, check their balance, view transaction history, and update their PIN. The system is built using Object‑Oriented Programming principles, with modular classes such as ATM, BankAccount, BankSystem, and Transaction to separate responsibilities and make the code maintainable. It demonstrates the use of encapsulation, constructors, collections like ArrayList, and secure user authentication.
